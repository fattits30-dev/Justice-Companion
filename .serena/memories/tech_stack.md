# Tech Stack

## Frontend

- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Desktop**: Electron
- **PWA**: Vite Plugin PWA
- **Testing**: Vitest, Playwright
- **Linting**: ESLint, TypeScript strict mode

## Backend

- **Framework**: FastAPI (Python)
- **Database**: SQLite (local) / PostgreSQL (cloud)
- **ORM**: SQLAlchemy 2.0
- **Authentication**: PassLib + Cryptography
- **Rate Limiting**: Custom implementation
- **Logging**: Winston + Python logging

## AI Service

- **Framework**: FastAPI (Python)
- **Models**: Hugging Face (privacy-first)
- **OCR**: Tesseract + Pillow
- **PDF Processing**: PyPDF + pdf2image
- **Fallback**: OpenAI API

## Development Tools

- **Linting**: Flake8, MyPy, Black (Python), ESLint (JS/TS)
- **Testing**: pytest, vitest, playwright
- **Security**: Snyk for vulnerability scanning
