# Justice Companion - Project Purpose

Justice Companion is a privacy-first AI-powered case management system designed specifically for UK legal matters. It provides comprehensive case management functionality while prioritizing user privacy through local AI processing and open-source models.

## Key Features

- Legal case management and document analysis
- AI-powered document analysis using Hugging Face models
- Privacy-first architecture with optional local model deployment
- Desktop application (Electron) with PWA support
- Multi-database support (SQLite for local, PostgreSQL for cloud)
- End-to-end encryption for sensitive legal data
