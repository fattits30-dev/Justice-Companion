## What Happened?

The better-sqlite3 native binary was compiled for Node.js v23.x but you're running Node.js v22.20.0.

**Technical Details:**

- Binary expects: NODE_MODULE_VERSION 139 (Node v23.x)
- Runtime provides: NODE_MODULE_VERSION 127 (Node v22.x)
- Result: Database initialization fails, cascading to 245 test failures

### 2. Fix ErrorBoundary Test (5 min)

**File:** `src/components/ErrorBoundary.test.tsx`
**Line:** ~Line 115 (search for "should display default message")

**Change:**

```typescript
// BEFORE:
screen.getByText('An unexpected error occurred');

// AFTER:
screen.getByText(/An unexpected error occurred/i);
```

**Why:** Text is split across DOM elements, exact match fails. Regex is flexible.

---

### 3. Fix SettingsView Test (10 min)

**File:** `src/features/settings/components/SettingsView.test.tsx`
**Line:** ~Line 222 (search for "should update profile on save")

**Change:**

```typescript
// BEFORE:
await user.clear(emailInput);

// AFTER:
await waitFor(() => {
  expect(emailInput).toBeEnabled();
});
await user.clear(emailInput);
```

**Why:** Input isn't focusable yet when clear() is called. Need to wait.

---
